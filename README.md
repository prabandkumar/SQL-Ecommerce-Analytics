# SQL E-Commerce Analytics

## Overview
Analyzed transactional e-commerce data using SQL to understand customer behavior, sales performance, and revenue distribution.

## Schema
- customers – customer details  
- products – product catalog and pricing  
- orders – order records  
- order_items – products per order with quantities  

## Analysis Performed
- Customers with and without orders  
- Repeat customers  
- Total and average order value  
- Product-wise sales and revenue  
- Order distribution across customers  

## Key Insights
- Not all registered customers place orders  
- A few products contribute most of the revenue  
- Repeat customers generate higher total spend  


## Tech Stack
- SQL (MySQL)

## Usage
1. Run `schema.sql`
2. Load `data.sql`
3. Execute queries from `queries.sql`
